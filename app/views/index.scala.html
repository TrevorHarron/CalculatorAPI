The Calculator API is a very simple project with a simple idea:
how can someone use Play 2.x to make a flexible API and
how can functional concepts be used to handle similar requests?

The result of this simple API where there are six functions: add, subtract(sub), multiply(mul), divide (div), modulo(mod), and power(pwr).

These fuctions are all POST requests where the the function accepts json
and then returns json.

For more information about these fuction calls go to /math
