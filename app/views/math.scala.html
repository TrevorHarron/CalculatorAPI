The only parameter needed for these function calls are a json with a "values" key pointing to a list of numbers (like "values":[1,2,4])

To make a sample function call with curl:
curl --header "Content-type:application/json" --request POST --data '{"values":[2,3]}' http://calcualtions-api-simple.herokuapp.com/math/sub
With the result:
'{"result":-1.0}'
